Installation

How to install:

  • 1. Unpack this archive in your wp-content/themes/ directory
  • 2. Upload the theme to the themes folder
  • 2. Upload the plugins to the plugin folder
  • 4. Go to Admin WordPress and select Design
  • 5. Select the Typo 3.17 WordPress Theme
  • 6. Configure the options
  • 7. You're done, have fun.

Customization:

  • - Open the theme folder BG and create your own background images
  • - Alignment classes: by default no post images are floated,
    • use alignleft, alignright or align center
  • - Open twitter.php, edit your Twitter RSS address
  • http://twitter.com/statuses/user_timeline/1487781.rss
  • - The Typo theme is register enabled, if you allow registering new users/authors, then the form in header template will appear, otherwise delete the link in nav.php
  • - Support for WP 2.7.* tags is built in, just tag your posts

Usable Plugins:

  • 1. WP Cache available at Uib
  • 2. Poll Plugin available at Lester Chan

Lead Image

Video post:

  • 1. Find an appropriate image, resize it (max 420px) and upload it through the post admin panel.
  • 2. Copy the image URL
  • 3. Scroll in post admin panel til CUSTOM FIELDS, select KEY from left menu
  • 4. Add the KEY lead_image into the KEY field
  • 5. Paste the image URL into the VALUE field
  • http://PATH TO IMAGE/uploads/2007/09/vj.gif
  • 6. Save
  • 7. Check your post for errors, if no errors "Publish".
